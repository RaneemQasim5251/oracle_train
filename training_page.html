Oracle E‑Business Suite – Training Page

English Instructions

Login Steps

  1. Open Firefox: Click the Firefox icon on your computer. The browser will open directly to the Oracle login page.
  2. Select language (optional): At the bottom of the login box, go to the Language field, click the small arrow, and choose Arabic or another preferred language.
  3. Enter your credentials: In the User Name field, type your username; in the Password field, type your password carefully.
  4. Log in: Click the Login button. If the information is correct, you will see the main Oracle page.

Navigating the System

On the right side of the main page, you will find a list of available applications. Click an item—for example, Purchase Order Details—to open it. You can also use the search box at the top to quickly locate any screen.

When you select an application, the system will launch the familiar Java window. If a security prompt appears, accept it to proceed.

Important: The core system remains unchanged—the same screens, tools and permissions. The updates affect only the interface and underlying databases; your workflows continue as before.

Adding to Favorites

  1. From the main page, click the Add to Favorites icon.
  2. Select the relevant responsibility from the Responsibility drop‑down menu and click Go.
  3. In the list of available functions, choose the function you want and click Add to Favorites.
  4. Scroll down to the Favorites section to confirm it has been added. You can reorder favorites or choose to open them in a new window.
  5. Click Apply to save your changes. Back on the main page, the new favorite will appear as an icon for quick access.

الإرشادات العربية

خطوات تسجيل الدخول

  1. فتح متصفح فايرفوكس: اضغط على أيقونة فايرفوكس في الجهاز. سيفتح المتصفح مباشرة صفحة تسجيل الدخول لنظام أوراكل.
  2. اختيار اللغة (اختياري): في أسفل مربع تسجيل الدخول نذهب إلى خانة اللغة، ثم نختار اللغة العربية أو أي لغة مفضلة.
  3. إدخال اسم المستخدم وكلمة السر: نكتب اسم المستخدم في خانة اسم المستخدم، ثم كلمة السر في خانة كلمة السر بدقة.
  4. الدخول: نضغط على زر تسجيل الدخول، فإذا كانت البيانات صحيحة تظهر الصفحة الرئيسية للنظام.

التنقل في النظام

في الجهة اليمنى من الصفحة الرئيسية تظهر قائمة التطبيقات. ننقر مثلاً على خيار تفاصيل أوامر الشراء لفتحه. كما يمكن استخدام خانة البحث في الأعلى للوصول السريع لأي شاشة.

عند اختيار أي تطبيق، يفتح النظام نافذة جافا المعروفة. قد تظهر رسالة أمنية؛ في هذه الحالة نوافق عليها للاستمرار.

تنبيه: يظل النظام كما هو: نفس الشاشات والأدوات والصلاحيات. التغيير فقط في واجهة الاستخدام وتحديث قواعد البيانات.

إضافة للمفضلة

  1. من الصفحة الرئيسية نضغط على أيقونة إضافة مفضلة.
  2. نختار المسؤولية المناسبة من القائمة المنسدلة ونضغط اذهب.
  3. نحدد الوظيفة المراد إضافتها ونضغط إضافة إلى المفضلة.
  4. ننزل إلى قسم المفضلة للتأكد من إضافتها، ويمكن ترتيبها للأعلى أو الأسفل عند الحاجة.
  5. نضغط تطبيق لحفظ التغييرات، وستظهر الوظيفة في الصفحة الرئيسية كأيقونة مفضلة.

Conclusion

The system’s functionality remains unchanged; only the interface and databases have been improved. We hope this guide helps you use the new interface with confidence.
